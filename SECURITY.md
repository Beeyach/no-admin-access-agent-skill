# Security Policy

## Supported version

Security fixes are applied to the latest release.

## Report a vulnerability

Do not publish credentials, private company policy, internal paths, device identifiers, or exploit details in a public issue. Use GitHub's private vulnerability reporting feature for this repository.

## Scope

The command checker detects known text patterns. It is not a sandbox, policy engine, antivirus tool, or proof that a command is safe. A script, package hook, alias, or executable can perform actions not visible in its command line.

The skill never authorizes privilege escalation or attempts to bypass managed-device controls.
